Detailed Description

Created by apolol92 on 20.08.15. Piecewise Aggregate Approximation(PAA) use a non-overlapping window of length W. It can be used for dimensionality reduction by replacing all DataPoints in the window with the window-mean. It also can be used to approximate a time series.
