Detailed Description

Created by apolol92 on 20.08.15. Represents a time series. A time series is a sequence of data points. There are two types of time series:

  • Regular time series: The spacing between every observation/data point is always the same
  • Irregular time series: The spacing between every observation/data point is different But these two types have much in common. For this reason we use an abstract parent class called TimeSeries.
